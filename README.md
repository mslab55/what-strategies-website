# What Strategies Website

A professional strategy consulting website built with vanilla HTML, CSS, and JavaScript.

## Overview

This site is designed in the style of leading political and policy strategy consulting firms — clean, minimalist, and authoritative. The design draws inspiration from firms like Teller Strategies and August Strategy Group, emphasizing professionalism, clarity, and credibility.

## Design Principles

- **Minimalist layout** — Single-column, vertically scrolling structure with generous whitespace
- **Neutral color palette** — White/off-white backgrounds, dark text, with navy or charcoal accents
- **Sans-serif typography** — Modern, readable fonts with strong size hierarchy for headlines vs. body copy
- **Professional tone** — Authoritative yet approachable messaging focused on expertise and partnership
- **Clear calls to action** — Prominent contact/get-in-touch prompts throughout

## Planned Sections

1. **Navigation** — Simple top bar with page links and a "Get in Touch" CTA button
2. **Hero** — Bold headline, brief tagline, and a primary call-to-action
3. **About** — Value proposition and firm background
4. **Services/Expertise** — Grid or card layout showcasing practice areas
5. **Team** — Headshots, names, and bios
6. **Contact** — Contact form and/or contact details
7. **Footer** — Social links, legal notices

## Tech Stack

- HTML5
- CSS3 (no framework — hand-written styles)
- Vanilla JavaScript
- Python (for any backend/tooling needs)

## Getting Started

No build tools required. Open `index.html` directly in a browser or serve locally:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.
